(b) How a single decoder reconstructs a partially aligned shared pair (φᵢ, ψᵢ)

(i) Ideal
Two dictionary columns recover each endpoint separately.
✓ strict recovery, ✓ cross-modal slot identity.
(ii) Naive single decoder — interference
φᵢ and ψᵢ share one decoder column; their pulls cancel onto the bisector.
Cause: interference — one shared column must reconstruct both atoms; the two pulls cancel onto the bisector.
✗ each endpoint off by α/2; recovered α inflated.
(iii) + prior alignment loss — forced merge
Lalign ties two latent slots together → extra slots collapse to one direction.
Cause: dimensionality reduction — the loss explicitly forces two latent slots to be identical, so multiple columns waste capacity on one merged direction.
✗ merge amplified; dictionary slots wasted.
