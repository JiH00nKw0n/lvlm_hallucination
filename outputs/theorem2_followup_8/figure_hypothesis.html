Theorem 2 on partially aligned shared atoms — hypothesis

Shared atoms $\boldsymbol{\phi}_i, \boldsymbol{\psi}_i$ are partially aligned (cosine $\alpha$, neither identical nor orthogonal). How does a single decoder recover them?
(a) Ideal reconstruction
Two SAE columns land exactly on $\boldsymbol{\phi}_i$ and $\boldsymbol{\psi}_i$.
Two dictionary columns $\mathbf{v}_1, \mathbf{v}_2$ recover each endpoint exactly. ✓ strict recovery, ✓ cross-modal slot identity (trivial if two slots bound to same shared index).
(b) Naive single decoder ($\mathcal{L}_{\mathrm{rec}}$ only)
Theorem 2 partitions $\{\boldsymbol{\phi}_i, \boldsymbol{\psi}_i\}$ into one group; top eigenvector lands on the bisector.
One column merges both shared atoms. Bisector 에 떨어진 $\mathbf{v}$ 는 $\boldsymbol{\phi}_i$ 도 $\boldsymbol{\psi}_i$ 도 각도 $\alpha/2$ 만큼 빗겨남. ✗ strict endpoint recall, recovered $\alpha_{\mathrm{proxy}}$ inflated.
(c) + prior alignment loss ($\mathcal{L}_{\mathrm{rec}} + \lambda \mathcal{L}_{\mathrm{align}}$)
$\mathcal{L}_{\mathrm{align}}$ pulls "shared-index $i$" latents to identity → extra columns collapse to bisector.
$\mathcal{L}_{\mathrm{align}}$ forces $\tilde z_{\mathrm I,i} \approx \tilde z_{\mathrm T,i}$, so both slots must fire identically for $\boldsymbol{\phi}_i$ and $\boldsymbol{\psi}_i$ → both columns collapse to the bisector. ✗ merging amplified, multiple dictionary slots wasted on one merged feature.
